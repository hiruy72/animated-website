## 1. OVERVIEW
--------------------------------------------------------------
This project recreates smooth animations, clip-path transitions,
video-based storytelling, and polished UI interactions.
I am using this fork for practice and experimentation.



## 2. FEATURES
--------------------------------------------------------------
- Scroll-triggered GSAP animations
- Clip-path geometric transitions
- 3D hover effects
- Video transitions
- Fully responsive layout
- Reusable component structure

## 3. TECH STACK
--------------------------------------------------------------
- React.js
- GSAP (GreenSock)
- Tailwind CSS
- Vite

## 4. SETUP
--------------------------------------------------------------
